# Vũ Portfolio Starter

Đây là project starter cho portfolio cá nhân của Vũ, host trên GitHub Pages.

## Cấu trúc

- `index.html` — trang chính, layout đơn giản: About, Links, Chat demo.
- `style.css` — style cơ bản, dark theme.
- `script.js` — logic khung chat demo (chưa kết nối API).

## Cách dùng

1. Mở folder này trong VSCode.
2. Chạy `git init`, commit, rồi push lên GitHub.
3. Bật GitHub Pages (branch `main`, root).
4. Sau khi ổn giao diện, có thể thêm backend riêng để kết nối OpenAI API.
